1.Installation:
  Extract epidemic_reporting.zip and place in android studio folder.


2. Connect Firebase Database:
   Select - Android Studio->Tools->Firebase->Authentication(provide firebase account)->Realtime Database(provide firebase account)

3. Create Disease Table name as "Current Disease" and add four disease name  mapping their key 1,2,3,4.

4. Generate APK:
   Select - Android Studio->Build->Generate Signed Bundle->APK->Select path
   ->Key store Password(12345678)->key alias(key0)->release->V1(Jar Signature)->V2(Full APK Signature)

5. install the APK in proper device. Hopefully its work properly.
